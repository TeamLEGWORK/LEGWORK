Installation
============
Dependencies
------------
- numpy >= 1.16
- scipy >= 1.5.0
- astropy >= 4.0
- numba >= 0.50
- matplotlib >= 3.3.2
- seaborn >= 0.11.1


from PyPI
---------
LEGWORK is available for installation on pypi. To install the latest stable version using ``pip`` use: ::

   pip install legwork
